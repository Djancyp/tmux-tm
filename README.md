# Tmux session starter

This script allows you 2 create a new session or select from open sessions.

## Installation

```bash
git clone https://github.com/Djancyp/tmux-tm
```
Than copy script to bin folder
```bash
sudo cp tm /usr/local/bin
```

## Usage

```bash
tm
```
